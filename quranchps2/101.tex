\textamh{ቢስሚላሂ አራህመኒ ራሂይም } &  بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ\\
\textamh{1.\  } &  ٱلْقَارِعَةُ ﴿١﴾\\
\textamh{2.\  } & مَا ٱلْقَارِعَةُ ﴿٢﴾\\
\textamh{3.\  } & وَمَآ أَدْرَىٰكَ مَا ٱلْقَارِعَةُ ﴿٣﴾\\
\textamh{4.\  } & يَوْمَ يَكُونُ ٱلنَّاسُ كَٱلْفَرَاشِ ٱلْمَبْثُوثِ ﴿٤﴾\\
\textamh{5.\  } & وَتَكُونُ ٱلْجِبَالُ كَٱلْعِهْنِ ٱلْمَنفُوشِ ﴿٥﴾\\
\textamh{6.\  } & فَأَمَّا مَن ثَقُلَتْ مَوَٟزِينُهُۥ ﴿٦﴾\\
\textamh{7.\  } & فَهُوَ فِى عِيشَةٍۢ رَّاضِيَةٍۢ ﴿٧﴾\\
\textamh{8.\  } & وَأَمَّا مَنْ خَفَّتْ مَوَٟزِينُهُۥ ﴿٨﴾\\
\textamh{9.\  } & فَأُمُّهُۥ هَاوِيَةٌۭ ﴿٩﴾\\
\textamh{10.\  } & وَمَآ أَدْرَىٰكَ مَا هِيَهْ ﴿١٠﴾\\
\textamh{11.\  } & نَارٌ حَامِيَةٌۢ ﴿١١﴾\\
\end{longtable}
