\textamh{\ \ \ \ \ \  ቢስሚላሂ አራህመኒ ራሂይም } &  بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ\\
\textamh{1.\  } &  وَيْلٌۭ لِّكُلِّ هُمَزَةٍۢ لُّمَزَةٍ ﴿١﴾\\
\textamh{2.\  } & ٱلَّذِى جَمَعَ مَالًۭا وَعَدَّدَهُۥ ﴿٢﴾\\
\textamh{3.\  } & يَحْسَبُ أَنَّ مَالَهُۥٓ أَخْلَدَهُۥ ﴿٣﴾\\
\textamh{4.\  } & كَلَّا ۖ لَيُنۢبَذَنَّ فِى ٱلْحُطَمَةِ ﴿٤﴾\\
\textamh{5.\  } & وَمَآ أَدْرَىٰكَ مَا ٱلْحُطَمَةُ ﴿٥﴾\\
\textamh{6.\  } & نَارُ ٱللَّهِ ٱلْمُوقَدَةُ ﴿٦﴾\\
\textamh{7.\  } & ٱلَّتِى تَطَّلِعُ عَلَى ٱلْأَفْـِٔدَةِ ﴿٧﴾\\
\textamh{8.\  } & إِنَّهَا عَلَيْهِم مُّؤْصَدَةٌۭ ﴿٨﴾\\
\textamh{9.\  } & فِى عَمَدٍۢ مُّمَدَّدَةٍۭ ﴿٩﴾\\
\end{longtable} \newpage
