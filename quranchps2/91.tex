\nopagebreak
\textamh{\ \ \ \ \ \  ቢስሚላሂ አራህመኒ ራሂይም } &  بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ\\
\textamh{1.\  } &  وَٱلشَّمْسِ وَضُحَىٰهَا ﴿١﴾\\
\textamh{2.\  } & وَٱلْقَمَرِ إِذَا تَلَىٰهَا ﴿٢﴾\\
\textamh{3.\  } & وَٱلنَّهَارِ إِذَا جَلَّىٰهَا ﴿٣﴾\\
\textamh{4.\  } & وَٱلَّيْلِ إِذَا يَغْشَىٰهَا ﴿٤﴾\\
\textamh{5.\  } & وَٱلسَّمَآءِ وَمَا بَنَىٰهَا ﴿٥﴾\\
\textamh{6.\  } & وَٱلْأَرْضِ وَمَا طَحَىٰهَا ﴿٦﴾\\
\textamh{7.\  } & وَنَفْسٍۢ وَمَا سَوَّىٰهَا ﴿٧﴾\\
\textamh{8.\  } & فَأَلْهَمَهَا فُجُورَهَا وَتَقْوَىٰهَا ﴿٨﴾\\
\textamh{9.\  } & قَدْ أَفْلَحَ مَن زَكَّىٰهَا ﴿٩﴾\\
\textamh{10.\  } & وَقَدْ خَابَ مَن دَسَّىٰهَا ﴿١٠﴾\\
\textamh{11.\  } & كَذَّبَتْ ثَمُودُ بِطَغْوَىٰهَآ ﴿١١﴾\\
\textamh{12.\  } & إِذِ ٱنۢبَعَثَ أَشْقَىٰهَا ﴿١٢﴾\\
\textamh{13.\  } & فَقَالَ لَهُمْ رَسُولُ ٱللَّهِ نَاقَةَ ٱللَّهِ وَسُقْيَـٰهَا ﴿١٣﴾\\
\textamh{14.\  } & فَكَذَّبُوهُ فَعَقَرُوهَا فَدَمْدَمَ عَلَيْهِمْ رَبُّهُم بِذَنۢبِهِمْ فَسَوَّىٰهَا ﴿١٤﴾\\
\textamh{15.\  } & وَلَا يَخَافُ عُقْبَٰهَا ﴿١٥﴾\\
\end{longtable} \newpage
