\textamh{\ \ \ \ \ \  ቢስሚላሂ አራህመኒ ራሂይም } &  بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ\\
\textamh{1.\  } &  وَٱلضُّحَىٰ ﴿١﴾\\
\textamh{2.\  } & وَٱلَّيْلِ إِذَا سَجَىٰ ﴿٢﴾\\
\textamh{3.\  } & مَا وَدَّعَكَ رَبُّكَ وَمَا قَلَىٰ ﴿٣﴾\\
\textamh{4.\  } & وَلَلْءَاخِرَةُ خَيْرٌۭ لَّكَ مِنَ ٱلْأُولَىٰ ﴿٤﴾\\
\textamh{5.\  } & وَلَسَوْفَ يُعْطِيكَ رَبُّكَ فَتَرْضَىٰٓ ﴿٥﴾\\
\textamh{6.\  } & أَلَمْ يَجِدْكَ يَتِيمًۭا فَـَٔاوَىٰ ﴿٦﴾\\
\textamh{7.\  } & وَوَجَدَكَ ضَآلًّۭا فَهَدَىٰ ﴿٧﴾\\
\textamh{8.\  } & وَوَجَدَكَ عَآئِلًۭا فَأَغْنَىٰ ﴿٨﴾\\
\textamh{9.\  } & فَأَمَّا ٱلْيَتِيمَ فَلَا تَقْهَرْ ﴿٩﴾\\
\textamh{10.\  } & وَأَمَّا ٱلسَّآئِلَ فَلَا تَنْهَرْ ﴿١٠﴾\\
\textamh{11.\  } & وَأَمَّا بِنِعْمَةِ رَبِّكَ فَحَدِّثْ ﴿١١﴾\\
\end{longtable} \newpage
