\textamh{ቢስሚላሂ አራህመኒ ራሂይም } &  بِسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ\\
\textamh{1.\  } & بِّسْمِ ٱللَّهِ ٱلرَّحْمَـٰنِ ٱلرَّحِيمِ وَٱلتِّينِ وَٱلزَّيْتُونِ ﴿١﴾\\
\textamh{2.\  } & وَطُورِ سِينِينَ ﴿٢﴾\\
\textamh{3.\  } & وَهَـٰذَا ٱلْبَلَدِ ٱلْأَمِينِ ﴿٣﴾\\
\textamh{4.\  } & لَقَدْ خَلَقْنَا ٱلْإِنسَـٰنَ فِىٓ أَحْسَنِ تَقْوِيمٍۢ ﴿٤﴾\\
\textamh{5.\  } & ثُمَّ رَدَدْنَـٰهُ أَسْفَلَ سَـٰفِلِينَ ﴿٥﴾\\
\textamh{6.\  } & إِلَّا ٱلَّذِينَ ءَامَنُوا۟ وَعَمِلُوا۟ ٱلصَّـٰلِحَـٰتِ فَلَهُمْ أَجْرٌ غَيْرُ مَمْنُونٍۢ ﴿٦﴾\\
\textamh{7.\  } & فَمَا يُكَذِّبُكَ بَعْدُ بِٱلدِّينِ ﴿٧﴾\\
\textamh{8.\  } & أَلَيْسَ ٱللَّهُ بِأَحْكَمِ ٱلْحَـٰكِمِينَ ﴿٨﴾\\
\end{longtable}
